导读 新增服务商 选择模型 验证对话

导读

Cherry Studio 更适合非工程同事使用。把服务商信息配置好后，就可以在一个客户端里切换模型和场景。

1. 新增服务商

1
打开设置

进入模型服务商配置。

2
新增自定义服务商

填入 Base URL 和 API Key。

3
保存配置

避免把真实密钥贴进公共文档。

2. 选择模型

场景推荐
日常问答选择速度较快的通用模型。
长文总结选择上下文能力更强的模型。
图片生成选择图像模型或使用绘图教程里的命令。

3. 验证对话

保存后新建一个会话，发送一句测试问题。如果返回异常，检查服务地址、密钥、模型名和网络连接。

测试提示词
请用三句话介绍这个模型配置是否已经可用，并给我一个后续测试建议。
